			<p>Слухи о кончине этого сайта несколько преувеличены. Вашему вниманию предлагается импровизированное видео, смонтированное из слайдов и аудио-записи доклада «Веб-шрифты vs. Шрифты для веба», прозвучавшего 8 декабря на ВСТ-встрече в Москве в рамках <a href="http://www.webhitech.ru/party/">WebHiTech Party</a>.</p>

<code>    overflow:hidden;</code>

<code>    position:absolute;</code>
<code>    top:0;</code>
<code>    left:0;</code>
<code>    width:title_width;</code>
<code>    height:title_height;</code>
